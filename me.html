About Me

저는 사용자 경험을 중심으로 제품을 구현하는 프론트엔드 개발자입니다. Flutter 기반 모바일 개발에서 시작해 React와 Next.js 기반 웹 개발까지 확장했고, 디자인 시스템, 상태관리 구조, 반응형 UI, 인터랙션 구현처럼 프론트엔드의 핵심 영역을 중심으로 경험을 쌓아왔습니다.

개발을 하면서 좋은 프론트엔드는 화면을 만드는 것만으로 완성되지 않는다고 느꼈습니다. 실제 사용자 흐름에 맞는 구조를 설계하고, API와 데이터 모델을 이해한 상태에서 UI를 설계해야 제품이 자연스럽게 동작한다고 생각합니다. 그래서 UI 개발과 함께 기획 문서 작성, API 구조 설계, 데이터 모델 설계, 서버사이드 결제 전환과 같은 백엔드 중심 플로우 설계도 직접 맡아왔습니다.

최근에는 Claude API와 MCP를 활용해 디자인, 코드 생성, 테스트, 문서화 흐름을 자동화하는 개발 파이프라인도 구축했습니다. 저는 프론트엔드 개발자로서 사용자 경험에 가장 가까운 위치에서 일하되, 제품이 실제로 만들어지고 확장되는 과정 전체를 이해하는 개발자이고자 합니다.
